## About / 关于

MediLink Shanghai connects international patients directly to top specialists at Shanghai's leading public tertiary hospitals — managing every detail from initial consultation to discharge.

沪医通帮助外籍患者直接对接上海公立三甲医院顶级专家，从首次咨询到出院随访，全程专业陪护。
